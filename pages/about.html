About

What even is CodeUtilities Scripts?

CodeUtilites Scripts, is a way for the player to code custom scripts to modify or add things to the game, for example a script that adds a /todo command that displays anything you put after on your screen, or a /ds command that gives you a debug stick, its up to your imagination!

How do I use CodeUtilities Scripts?

To use CodeUtilites Scripts you need to have the mod here, when you are done and you launched Minecraft type /scripts, here you can create, edit, remove and reload scripts.

I found a bug! Where do I report it?

If you found a bug, you can report it on the Issues page on Github or in our Discord Server.
