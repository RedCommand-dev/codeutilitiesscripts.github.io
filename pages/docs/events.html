Events

In the order specified in here.

SendChat

Called when the player sends a message into the chat.

Sets the message variable to the message sent.

ReceiveChat

Called when the player receives a message from the server.

Sets the message variable to the message received.

KeyPress

Called when the player presses a key.

Sets the key variable to the key pressed.

KeyRelease

Called when the player releases a key.

Sets the key variable to the key released.

Tick

Called every tick.

Init

Called when the script is loaded.

RegisterCmds

Called whenever commands are supposed to be registered.

SendCommand

Called when the player sends a command.

Sets the command variable to the entire command sent with the slash removed. Only meant for commands that are registered via scripts.

Finalize

Called when the script is unloaded.

RenderGui

"Called when a gui is being rendered.

Sets screenWidth and screenHeight
